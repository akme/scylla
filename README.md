# scylla
#### Start cluster:
```bash
# ./manage.py -init
```
#### Add nodes to initial cluster:
```bash
# ./manage.py -add
```
#### Add nodes by IP (all ports should be available):
```bash
# ./manage.py -s <IP>
```
#### Get cluster state:
```bash
# ./manage.py -status
```
#### Access cluster:
```bash
# ./manage.py -connect
```
